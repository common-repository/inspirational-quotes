== Description ==

Add some inspiration to your blog! This super-simple WordPress plugin displays an inspirational quote in your blog's sidebar. Now with over 120 hand selected quotes, here are some of this plugin's features:

= Features =

* **Simplicity -** No complex installation or settings; just add the widget to your sidebar and go - it's that simple.
* **Quality Quotes -** Comes packed with 120+ hand-selected quotes - more are added regularly!
* **2 Display Options -** Choose to update the quote daily or have it select a random quote on each page view.


== Installation ==

1. Upload the file `inspirationalquotes.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and drag 'Daily Inspiration' to your sidebar
4. Update the settings in the widget: Title, When to Update, etc.


== Frequently Asked Questions ==

Settings can be edited from within the widget itself, here are the options:

= "Title" Option: =
This option will change the front-end title of your widget, it defaults to "Inspirational Quote"

= "Update Quotes" Option: =
This option allows you to either update the quote once per day, or have it display a random quote each time your blog is viewed. 

= "Give Us Credit" Option: =
This option helps us promote our widget and we hope you'll keep it. However, to easily remove this line, simply choose "no" under the "Give Us Credit" option.
